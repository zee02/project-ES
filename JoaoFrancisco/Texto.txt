Gestão de Empréstimos - Caminho Principal

1 - O utilizador seleciona a opção "Gestão de Empréstimos" na janela "Categorias".
2 - O sistema mostra a janela "Restrições de Empréstimos".
3 - O utilizador ao selecionar o botão “Valores de Dívida” o sistema mostra a janela "Valores de Dívida" com a respetiva informação.
4 - O utilizador ao selecionar o botão “Anuidade” o sistema mostra a janela "Anuidade" com a respetiva informação.
5 - O utilizador ao selecionar o botão “Restrições” o sistema mostra a janela "Restrições" com a respetiva informação.



Visualizar Multas - Caminho Principal

1 - O utilizador seleciona a opção “Multas” na janela "Gestão de Empréstimos".
2 - O sistema mostra a janela "Gerenciar Multas".
3 - O utilizador preenche com ID do sócio 
4 - O sistema verifica que os dados estão corretos, e mostra uma janela "Visualizar Multas" .


Caminhos alternativos

4.1 - Caso o ID seja inexistente mostra uma mensagem de erro.





Empréstimos - Caminho Principal

1 - O utilizador seleciona a opção "Empréstimos" na janela "Gestão de Empréstimos".
2 - O sistema mostra a janela "Registar um Novo Empréstimo".
3 - O utilizador preenche com o ID
4 - O sistema  verifica que os dados estão corretos, e mostra a janela “Realizar empréstimo”
5 - O utilizador preenche os dados do sócio.
6 - O sistema verifica que os dados estão corretos e guarda os dados


Caminhos alternativos

4.1 - Caso esteja tudo bem aparece uma mensagem de sucesso
4.1.1- Caso o ID seja inexistente mostra uma janela de erro.
4.1.2- Caso o Sócio esteja com o limite ultrapassado de empréstimos será mostrado uma janela de erro
4.1.3- Caso o Sócio tenha alguma multa pendente o sistema mostra uma janela de erro
4.1.4 - Caso os dados estejam mal preenchidos o sistema mostra uma janela de erro
6.1 - O sistema redireciona o utilizador para a pagina Registar um Novo Empréstimo".
 


Devolução - Caminho Principal

1 - O utilizador seleciona a opção "Devolução" na janela "Gestão de Empréstimos".
2 - O sistema mostra a janela "Registar uma Devolução".
3 - O utilizador preenche com o ID do sócio
4 - O sistema verifica o ID do sócio e mostra uma janela “Realizar Devolução”
5 - O utilizador preenche os dados do sócio.
6 - O sistema verifica que os dados estão corretos e guarda os dados

Caminhos alternativos

4.1 - Caso o ID seja inexistente mostra uma janela de erro.
4.1.1 - Caso os dados estejam mal preenchidos o sistema mostra uma janela de erro
6.1 - O sistema redireciona o utilizador para “Registar uma Devolução“ 





Catálogo de Livros - Caminho Principal

1 - O utilizador seleciona a opção "Gestão de Livros e Aquisições" na janela "Categorias".
2 - O sistema mostra a janela "Gestão de Livros e Aquisições".
2 - O utilizador seleciona "Catálogo de Livros"
3 - O sistema mostra a janela "Catálogo de Livros".
4 - O utilizador seleciona "Visualizar"
5 - O sistema mostra a janela "Catálogo Livro".
6 - O utilizador seleciona "Visualizar"
7 - O sistema mostra a janela "Visualizar Livro".
8 - O utilizador preenche os dados do sócio.
9 - O sistema verifica que os dados estão corretos e guarda os dados


Caminhos alternativos

9.1 - O sistema redireciona o utilizador para “Catálogo de Livros “ 
9.1.1 - Caso os dados estejam mal preenchidos o sistema mostra uma janela de erro







Adicionar Livro - Caminho Principal

1 - O utilizador seleciona a opção "Gestão de Livros e Aquisições" na janela "Categorias".
2 - O sistema mostra a janela "Gestão de Livros e Aquisições".
3 - O utilizador seleciona "Aquisições"
4 - O sistema mostra a janela "Aquisições".
5-  O utilizador seleciona "+"
6 - O sistema mostra a janela "Novo Livro".
7 - O utilizador preenche os respetivos dados.
8 - O sistema verifica que os dados estão corretos e guarda os dados

Caminhos alternativos

7.1 - O sistema mostra um POP-UP de confirmação se o utilizador deseja adicionar o livro.
7.1.1 - Se os dados estiverem corretos aparecerá uma mensagem de sucesso
7.1.2 - O sistema redireciona o utilizador para “Aquisições “ 
7.1.3 - Caso os dados estejam mal preenchidos o sistema mostra uma janela de erro




Gerenciar  Livro - Caminho Principal

1 - O utilizador seleciona a opção "Gestão de Livros e Aquisições" na janela "Categorias".
2 - O sistema mostra a janela "Gestão de Livros e Aquisições".
2 - O utilizador seleciona "Aquisições"
3 - O sistema mostra a janela "Aquisições".
4 - O utilizador seleciona "Gerenciar"
5 - O sistema mostra a janela "Gerenciar  Livro".
6 - O utilizador preenche os respetivos dados que deseja alterar.
7 - O sistema verifica que os dados estão corretos e guarda os dados

Caminhos alternativos

6.1 - O sistema mostra um POP-UP de confirmação se o utilizador deseja guardar os dados.
6.1.1 - Se os dados estiverem corretos aparecerá uma mensagem de sucesso
6.1.2 - O sistema redireciona o utilizador para “ Aquisições “ 
6.1.3 - Caso os dados estejam mal preenchidos o sistema mostra uma janela de erro
